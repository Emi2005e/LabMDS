# Specificații aplicație TODO list
Vreau o aplicație Python de tip REPL (Command Line Interface) care să ruleze într-o buclă infinită. Aplicația trebuie să accepte următoarele comenzi de la tastatură:
1. `add <text>` - Creează o sarcină nouă.
2. `done <id>` - Marchează o sarcină ca fiind rezolvată.
3. `delete <id>` - Șterge o sarcină din listă.
4. `list` - Afișează toate sarcinile (arătând clar care sunt rezolvate și care nu).
5. `exit` - Oprește execuția programului.

Aplicația trebuie să gestioneze erorile (de exemplu, dacă introduc un ID inexistent) și să afișeze mesaje clare pentru utilizator. Salvează codul într-un fișier numit `todo.py`.
